# Sentiment Analysis on Social Media (Twitter)

## About
An NLP-based sentiment analysis system that classifies tweets as positive, negative, or neutral. Built using Python and machine learning as part of B.Tech CSE at Graphic Era University (2024).

## How It Works
1. Collected 10,000+ tweets in real-time using Tweepy API
2. Applied NLP preprocessing — tokenization, stopword removal, lemmatization
3. Extracted features from text data
4. Trained and compared multiple classification models including Logistic Regression
5. Achieved 94% accuracy on test data

## Results
- **Accuracy: 94%**
- Dataset: 10,000+ tweets
- Models compared: Logistic Regression, Naive Bayes, SVM
- Best performing model selected after evaluation

## Tech Stack
- Python
- NLTK (Natural Language Processing)
- Scikit-learn (ML Models)
- Tweepy API (Data Collection)
- Pandas, NumPy

## Key Features
- Real-time tweet collection
- NLP preprocessing pipeline
- Multi-model comparison
- Sentiment classification: Positive / Negative / Neutral
